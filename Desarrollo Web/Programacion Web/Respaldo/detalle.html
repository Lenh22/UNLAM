Valor: $10000

Descripción del curso: Hola y bienvenido a este curso.
Si es un principiante absoluto, el desarrollo web puede ser un desafío; este curso lo guiará y le brindará suficientes detalles para que se sienta cómodo y aprenda los temas correctamente. Debe sentirse como en casa si es un estudiante que está aprendiendo desarrollo web (por su cuenta o como parte de una clase), un profesor que busca materiales de clase, un aficionado, o alguien que solo quiera comprender más sobre cómo funcionan las tecnologías web.
El objetivo de este curso no es llevarlo de "principiante" a "experto", sino llevarlo de "principiante" a "cómodo para que se sienta seguro de experimentar más.

Requisitos previos: Cualquier PC con Windows o Linux

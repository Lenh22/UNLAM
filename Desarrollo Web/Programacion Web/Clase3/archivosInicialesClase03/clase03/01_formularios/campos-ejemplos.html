Campos y Atributos de Formularios

Select y Option

El elemento select va a tener un name declarado, dentro de este, los options son todas las opciones que se nos van a desplegar para poder elegir, este elemento va a tener el atributo value definido que es el valor que se va a enviar atraves de este formulario.

Textarea

El elemento textarea define un campo de entrada de varias líneas, el atributo rows especifica el número de líneas visibles en un área de texto y el atributo cols especifica el ancho

Output

El elemento output nos permite ver calculos en vivo, dependiendo de los valores que se manejen.

Tipos de inputs

Text

El type con valor text sirve para el ingreso de datos alfanumericos

Number

El type con valor number sirve para el ingreso de datos numericos, puedo establecer min y max, para validar numeros validos

Radio

Sirve para solo darle la posibilidad al usuario de seleccionar una sola opción, todos los inputs del mismo grupo deben tener el mismo name pero distinto value

Hombre Mujer Otro

Checkbox

Sirve para solo darle la posibilidad al usuario de seleccionar varias de opciones, todos los inputs deben tener distintos name y distintos value

Qué lenguajes corren del lado del cliente?

Password

Sirve para el ingreso de contraseñas, evitando la visualización de la misma

Email

Sirve para el ingreso de correos electronicos, este tipo de input nos da automaticamente una validación del tipo de correo electronico mientras estamos completando el campo.

Tel

Sirve para el ingreso de numeros telefonicos, la mejor de sus cualidades es que cuando nos encontramos a nivel mobile, el teclado que nos aparece es el telefonico.

Fecha

Sirve para el ingreso de fechas y nos despliega un calendario

Color

Sirve para el ingreso de colores y nos despliega una paleta de colores

URL

Valida que las direcciones ingresadas sean url validas

Datalist

Sirve para especificar una lista predefinida de elementos para un campo input

Atributos

Autofocus

El atributo autofocus permite que una vez cargada la pagina, el campo al que se le asigna sea el que tenga el cursor

Placeholder

El atributo placeholder se muestra dentro del campo

Required

El atributo required sirve para identificar los campos que son obligatorios

Value

Si declaramos el atributo value, este sera el contenido por defecto del campo

Pattern

Sirve para validar un campo con una expresion regular, en este caso, valida que la patente, tenga 3 letras mayusculas, seguidas por 3 números
